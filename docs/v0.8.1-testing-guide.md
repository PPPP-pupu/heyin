# Heyin v0.8.1 — 多人内测指南

## 准备工作

### 1. 确认 EdgeOne 部署就绪

打开 `https://heyin-rtgvltyz.edgeone.cool`，确认首页正常显示。

### 2. 创建 2 个测试项目

**项目 A（简单版）：**

| 字段 | 填什么 |
|---|---|
| Title | 十年大合唱（A组） |
| Song Name | 十年 |
| Lyrics | 如果那两个字没有颤抖<br>我不会发现我难受<br>怎么说出口 |
| Voices per Line | 2 |

**项目 B（稍长版）：**

| 字段 | 填什么 |
|---|---|
| Title | 十年大合唱（B组） |
| Song Name | 十年 |
| Lyrics | 如果那两个字没有颤抖<br>我不会发现我难受<br>怎么说出口也不过是分手<br>如果对于明天没有要求<br>牵牵手就像旅游 |
| Voices per Line | 3 |

### 3. 获取 join 链接

创建完项目后，点 **Share** 按钮，复制链接。

### 4. 确认 owner 链接留存

地址栏里有 `?owner=xxx` —— 这是你的**管理链接**，不要发给测试者。

## 发给测试者（复制即用）

---

嗨！我做了一个多人合唱小工具，请你帮我测一下，**2 分钟**就够。

👉 打开这个链接：
【粘贴 join 链接】

然后照着做：

**第 1 步** — 输入你的昵称（比如「小雨」「阿Ken」），点「Join Chorus」

**第 2 步** — 点一个**灰色的空圈圈**（选一句你想唱的歌词）

**第 3 步** — 浏览器问你要麦克风，点「允许」

**第 4 步** — 点「Start Recording」，对着手机唱那句歌词

**第 5 步** — 唱完点红色「Stop Recording」

**第 6 步** — 可以点播放按钮试听，满意了就点「Submit」

**第 7 步** — 回到页面上，你刚才选的圈圈变绿了，点一下听自己的声音 🎉

不用唱得好！我就是想看看这个流程顺不顺。
